# Code vs. Manual — the bidirectional diff

**Status:** Derived 2026-07-11, **before** the end-to-end walk. Nothing here is a
walk result; everything here is a static reconciliation of what the framework
**says** against what the framework **does**.
**Companion:** [`CHECKLIST.md`](./CHECKLIST.md) — the acceptance checklist for the
walk. Every `UNENFORCED (declared only)` item there cites a `CM-*` row here.

**The governing rule:** *"The gate scripts are authoritative, prose guides
describe them and may lag — trust the scripts"* (`CLAUDE.md` § ENFORCEMENT —
SOURCE OF TRUTH). **Where prose and code disagree, the code wins, and this
document says so explicitly in every row.**
